Agent bridge

Install one MCP server, then let local agents query DingDong for summaries or load full resources only when needed.

{
  "mcpServers": {
    "dingdong": {
      "command": "/Applications/DingDong.app/Contents/MacOS/dingdong-mcp"
    }
  }

Shared library

Prompts, skills, MCP servers, knowledge references, and clipboard records live in one searchable local store.

Clipboard that behaves

Open with ⌘⇧V, preview with space, paste with return, and keep sensitive records out of agent context by default.

Local by default

The app listens on 127.0.0.1. Network access is only used for checking the public release JSON.
